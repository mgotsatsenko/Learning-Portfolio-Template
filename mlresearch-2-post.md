### Machine Learning Research

1. Explain AT LEAST TWO of the seven steps of Machine Learning as you would explain to a friend that
knows nothing about Machine Learning.
Two of the seven steps of Machine Learning that are easy to grasp are Gathering Data and Training. To prepare for a Machine Learning we first need to gather data for the specific task you have. Let's say based on 100 year historical data, you want ML to predict when will be the next big war and what will be the cause. You would get that data and then feed it, based on this data Machine Learning will be trained to identify patterns and give the best possible outcomes.   
2. What application of Machine Learning did you discover while researching?
When trying to research the applications of machine learning in identifying the next global war, I discovered that it is capable of using threat detection to predict a conflict based on mapping geopolitical instability and monitoring military movements. It can do so by also using satellite data in real time. 
a. How does Machine Learning improve or replace existing solutions?
It definitely improves threat detection by being quicker in covering large areas versus using a lot of man power to try and detect something by spying in areas like rainforest or rugged terrains.
b. Is the application in development or actively being used by companies/institutions?
Military Industrial complex is already using Machine Learning in the light of current events. There is a Maven Project that collects and analysis huge amounts of data collected from drones and satellites.
